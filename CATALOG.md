# fish

fish is a smart and user-friendly command line shell for Linux, macOS, and the rest of the family. It works out of the box with autosuggestions from your history, syntax highlighting as you type, and rich tab completions derived from man pages — no configuration required. Its scripting language is consistent and readable, with built-ins like `math` and `string` for everyday shell work.

## What's included

- **fish** — the fish command line shell (interactive prompt and `fish -c "..."` script runner)
